# Ola Dopemu – Premium Investor-Style GitHub Pages Bio Site

This package contains a premium one-page static website prepared for GitHub Pages.

## Files
- `index.html` – main page
- `styles.css` – premium investor-style visual design
- `.nojekyll` – tells GitHub Pages not to process the site with Jekyll
- `assets/headshot.jpg` – supplied headshot image prepared for the page
- `assets/og-image.svg` – social share image

## Quick publish steps
1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Make sure `index.html` sits in the repository root.
4. In GitHub: **Settings → Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select your branch (usually `main`) and folder **/** (root), then save.
7. GitHub Pages will publish your site.

## Before publishing
- Replace `https://your-username.github.io/` in the Open Graph URL meta tag.
- If you later add a custom domain, update the Open Graph URL accordingly.
